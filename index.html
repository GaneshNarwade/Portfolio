Projects

SpaceX Clone

SpaceXClone, is a captivating web app inspired by SpaceX's mission control. Its frontend, crafted with HTML5, CSS3, JavaScript, Ajax, Bootstrap, and Styled-components, delivers a visually stunning and engaging user experience. To enhance user experience further, SpaceXClone integrates Stripe payment processing, offering secure and convenient transactions for space-related services and merchandise. Whether users are tracking launches, exploring space missions, or purchasing SpaceX merchandise, SpaceXClone delivers a seamless and immersive experience at every step.




WeCare

WeCare, built in Java and employing the MVC architecture, efficiently handles hospital operations. It separates into Model (data and logic), View (user interface), and Controller (input handling), offering a modular and user-friendly solution for tasks like patient management, appointment scheduling, admin controls etc.




Path-Finder

Developed an interactive web-based visualizer using HTML, CSS, and JavaScript, offering insights into path finding algorithms like BFS, DFS, Dijkstra's, A*, and more.
Enhanced the project's versatility by incorporating Prim's algorithm with Euclidean distance heuristic, enabling users to explore minimum spanning trees, while also providing a clear visualization of topological sorting.
